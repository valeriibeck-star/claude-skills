# $200 Price Point — Validation Test
**Date:** 2026-03-29
**Category:** pricing
**Status:** UNTESTED — no calls made yet

## Strategy
- $200 for a complete static site (normally $5K-10K at agencies)
- Positioned as "building my portfolio in Miami, keeping it cheap on purpose"
- Zero risk pitch: "don't like it, don't pay"
- Payment via Stripe link (card/Apple Pay)

## Unit Economics
- API cost per site: ~$15
- Calling cost per prospect: ~$0.25
- Margin at $200: ~90%
- Breakeven: 1 sale covers ~13 site builds

## Hypothesis
$200 is impulse-buy territory for a plumber making $500K+/year whose site is broken. It removes the "too expensive" objection entirely. Risk: might be SO cheap it triggers "scam" suspicion.

## Counter to "too cheap" objection
"No catch. I need portfolio pieces and testimonials. You get a site, I get a case study."

## Planned Price Escalation
- First 5-10 clients: $200 (validation)
- Next 10: $500-1,000 (proven pipeline)
- Mature: $1,500-3,000 (established portfolio)

## Takeaway
Test at $200 first. If close rate is >10%, the model works and price can increase. If close rate is 0%, the issue is probably the approach, not the price.

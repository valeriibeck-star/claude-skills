# Two-Font Hierarchy Kills Template Feel
**Date:** 2026-03-29
**Category:** technique
**Related:** memory/clients/titan-plumbing/site.md

## What
Using Plus Jakarta Sans (700-900 weight) for headings and Inter for body text. The display font only goes on hero title, section titles, CTA banner.

## Result
V1 with single font (Inter only) scored 2.5/4 on Typography. V2 with Plus Jakarta Sans added scored 3.0/4. The GSD auditor specifically called it "the single biggest improvement from V1."

## Why It Works
Single-font sites look like templates because there's no visual separation between heading and body content. A display font at heavier weights creates instant hierarchy.

## Takeaway
Always use two font families. Display font for headlines (700+), clean sans-serif for body. Load both from Google Fonts with display=swap.
